## Scope

- **Framework:** NIST Cybersecurity Framework (CSF) — Identify Function
- **Analysis Type:** Risk, Control, and Impact (RCI)
- **Incident Context:** Security event with escalation due to preventable control gaps

---

## Key Findings

### Priority 1 — Immediate Action Required
- **Access Controls & Asset Validation:** Weak user group policies and unvalidated assets expanded the attack surface
- **Workforce Readiness:** Limited security training and unclear responsibilities increased human error and slowed response
- **Network Segmentation & Endpoint Protection:** Absence of segmentation allowed unrestricted lateral threat movement

### Priority 2 — Near-Term Improvements
- **Network Edge Detection & Data Protection:** Outdated IDS/IPS rules and weak encryption increased exposure risk

### Priority 3 — Ongoing Maturity
- **Logging, Monitoring & Audit Oversight:** Logs existed but were not actively reviewed, delaying detection and response
